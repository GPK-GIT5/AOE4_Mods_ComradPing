# Design Notes

Official AoE4 campaign pattern analysis and design-level reference material.

## Subtrees

| Folder | Covers |
|---|---|
| [campaigns/](campaigns/) | **Official campaign design notes** (abbasid, angevin, challenges, hundred, mongol, rogue, russia, salisbury) — one README per campaign with mission overview, shared patterns, and Onslaught-relevant analysis |

## Important distinction

`campaigns/` here documents **official singleplayer campaigns** shipped with AoE4, analyzed for patterns applicable to custom mod design. Our **custom coop scenarios** live under [../scenarios/](../scenarios/).

Each campaign folder contains exactly one `README.md` — a high-level overview of the campaign's missions, recurring mechanics, and relevance to the Onslaught project.
