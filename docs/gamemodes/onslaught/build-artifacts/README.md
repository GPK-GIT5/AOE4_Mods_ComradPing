# Onslaught Build Artifacts

The `Gamemodes/Onslaught/cache/` and `Gamemodes/Onslaught/archives/` folders from the private workspace are not present in this public repository.

## Folders covered

### `cache/`

Content-addressed build cache generated by the AoE IV Content Editor during each mod build. Contains thousands of SHA-hash-named subdirectories of intermediate build assets. Regenerated completely on each build.

### `archives/`

Compiled binary mod distribution artifacts.

```
archives/
└── CBA_Custom_v1f.sga   # Compiled, packaged mod — output of the build pipeline
```

## Integration points

| Artifact | Role |
|---|---|
| `CBA_Custom_v1f.sga` | The distributable mod file loaded by the AoE IV game client |
| `cache/` | Local build accelerator used only by the Content Editor |

## Local regeneration

Both are regenerated by opening [`Gamemodes/Onslaught/CBA Custom v1f.aoe4mod`](../../../../Gamemodes/Onslaught/CBA%20Custom%20v1f.aoe4mod) in the AoE IV Content Editor and running a build against the published source in [`Gamemodes/Onslaught/assets/`](../../../../Gamemodes/Onslaught/assets/).
